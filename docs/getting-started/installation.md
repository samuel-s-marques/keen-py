# Installation

Pre-requirements:
- Python 3.10 or higher
- pip
- Git

To install Keen, you need to clone the project with Git or download the latest release from the [releases page](https://github.com/samuel-s-marques/keen-py/releases).

```bash
git clone https://github.com/samuel-s-marques/keen-py.git
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Run Keen:

```bash
python keen.py
```
